# TKO Bot 🃏

A Discord card-collecting bot — pull anime & game character cards, build your collection, trade with others, and earn shards from duplicates.

---

## ✅ Setup (5 minutes)

### 1. Install Node.js 18+
Download from https://nodejs.org

### 2. Create your Discord Bot
1. Go to https://discord.com/developers/applications
2. Click **New Application** → name it "TKO Bot"
3. Go to the **Bot** tab → **Add Bot**
4. Under **Privileged Gateway Intents**, enable:
   - ✅ **Message Content Intent**
   - ✅ **Server Members Intent**
5. Copy your **Bot Token**

### 3. Invite the Bot to your Server
1. Go to **OAuth2 → URL Generator**
2. Scopes: `bot`
3. Permissions: `Send Messages`, `Read Message History`, `Embed Links`
4. Open the URL and invite the bot

### 4. Add your token
Rename `.env.example` to `.env` and paste your token:
```
DISCORD_TOKEN=paste_your_token_here
```

### 5. Install & Run
```bash
npm install
npm start
```

## 🎮 Commands

| Command | Description |
|---|---|
| `!tko pull` | Pull a random card (30s cooldown) |
| `!tko inventory` | View your card collection |
| `!tko inventory @user` | View another user's collection |
| `!tko card <cardId>` | Inspect a specific card |
| `!tko shards` | Check your shard balance 💎 |
| `!tko shards @user` | Check another user's shards |
| `!tko trade @user <cardId> <shardsAsking>` | Offer a card for shards |
| `!tko accept <tradeId>` | Accept a trade offer |
| `!tko decline <tradeId>` | Decline a trade offer |
| `!tko trades` | View trade offers sent to you |
| `!tko help` | Show all commands |

## 🔧 Customising Pull Rates

Open `config.js` and edit `PULL_RATES` — just make sure all values add up to **100**:

```js
PULL_RATES: {
  R:  45,   // 45%
  E:  28,   // 28%
  L:  15,   // 15%
  MY:  7,   //  7%
  UR:  4,   //  4%
  LT:  1,   //  1%
},
```

## ➕ Adding New Cards

Open `cards.js` and add a new entry to the `CARDS` array:

```js
{ id: 'my_card_ur', name: 'My Character', series: 'My Series', rarity: 'UR', desc: 'Flavour text here.' },
```

- `id` — unique snake_case string (used in trades & inventory)
- `rarity` — must be one of: `R`, `E`, `L`, `MY`, `UR`, `LT`
- `desc` — short flavour text shown in embeds
- `image` — *(optional)* image URL shown as thumbnail in the card embed

## 📁 File Structure

```

├── config.js       ← Pull rates, shard values, rarity display
├── inventory.js    ← JSON persistence (cards + shards)
├── trades.js       ← In-memory trade manager (5 min expiry)

├── .env.example    ← Template
├── package.json
└── data/
    └── inventory.json   ← Auto-created on first pull
```
